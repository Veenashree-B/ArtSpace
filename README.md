# ArtSpace App 🖼️

A refined Android application built as an interactive gallery experience. This project demonstrates proficiency in building modern, declarative user interfaces and handling state within Android development. 

## ✨ Features
* **Interactive Gallery:** Allows users to smoothly navigate forward and backward through a curated collection of artworks.
* **Modern UI Architecture:** Developed using **Jetpack Compose**, implementing a clean and modular component-based structure.
* **State Management:** Effectively utilizes Compose state handling (`remember`, `mutableIntStateOf`) to dynamically update the UI based on user interaction.
* **Material Design:** Incorporates Material 3 components for a polished, accessible, and responsive user experience. 

## 🛠️ Tech Stack
* **Language:** Kotlin
* **UI Toolkit:** Jetpack Compose
* **Design System:** Material Design 3
* **Build System:** Gradle (Version Catalog enabled)
